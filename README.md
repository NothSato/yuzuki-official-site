# Sanity Blog

プロジェクトID: `hg2k8me9` でSanityブログサイトが完成しました！

## 起動方法

開発サーバーは既に起動中です：
- **ブログサイト**: http://localhost:3000
- **Sanity Studio**: http://localhost:3000/studio

## 使い方

1. **Sanity Studioでコンテンツ作成**
   - http://localhost:3000/studio にアクセス
   - 「Category」でカテゴリーを作成
   - 「Tag」でタグを作成  
   - 「Post」で記事を作成

2. **ブログサイトで確認**
   - http://localhost:3000 で記事一覧を確認
   - 個別記事ページでコンテンツを確認

## 機能

✅ 記事投稿・編集機能
✅ カテゴリー・タグ管理
✅ レスポンシブデザイン
